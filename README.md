# Calc II Trainer

An educational game designed to help students strengthen their Calculus II integration skills by practicing one of the most important parts of solving integrals: **recognizing which integration technique to use**. Instead of repeatedly solving the same textbook problems, players are presented with randomly generated integrals that require them to identify the correct strategy, helping build intuition and pattern recognition through repetition.

---

## Features

- Randomly generated integration problems for virtually unlimited practice
- Built-in tutorial explaining each integration technique and common clues
- Lightning Round mode for timed challenges
- Endless Mode for continuous practice
- Focus on developing pattern recognition and strategy selection

---

## Integration Techniques

The game currently includes practice for:

- U-Substitution
- Integration by Parts
- Trigonometric Integrals
- Trigonometric Substitution
- Partial Fraction Decomposition

---

## Why I Built This

One of the most difficult aspects of Calculus II is determining **which technique** should be applied before actually solving an integral or determining convergence/divergence of a series. While there are many resources for practicing calculations, there are fewer that focus specifically on developing this recognition skill.

Calc II Trainer was created to provide students with unlimited randomized practice so they can build confidence in identifying appropriate strategies through repetition.
